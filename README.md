# Dave
Dave is a school project that I originally created with MIT App Inventor and have fully filled out using pygame
    Both versions of the game were for a final project

# How to Play
Dave has been recognized for being a fat man (despite being a stick figure) and he wants to change that, so every day he's gonna try and Jog as far as he can!
- Each step you take will thrust you forward slightly
- Dave has limited stamina, he can only take so many steps at once
- There are things in the way of Dave's run, therefore he has to jump over them. But, this takes a lot more stamina.
- Before his jog, he thrust himself off of his recliner, this gives him a speed boost at the beginning. The more powerful the chair, the faster he goes!
- The higher the caffine dosage, the more stanmina he has!
- Dave's shoes arent great to begin with, but he can upgrade them to go faster with every step!
- A handy bomb and give Dave a little extra boost
- The teleporter will breifly do the walking for Dave! A breif moment of immunity.
- A bottle of 30 second enegry will give dave 30 stamnina points per use!

# Controls
## Main Game
SPACE - Take a step
SHIFT/UP - Jump
Z - Use Bomb
X - Use Teleporter
C - Use 30 Second Enegry

## Results Screen
RIGHT/ENTER - Open Shop
LEFT/ESC/BACKSPACE - Close Game
SPACE - Run again

## Shop
Up/Down - Scroll options
Left/Right - Change the level of an upgrade (if applicable)
SPACE - Buy upgrade
ENTER - Run again
